GitHub图床使用说明

创建 GitHub 仓库

创建一个用来存储图片的GitHub仓库

获取 GitHub Token

创建一个有repo权限的GitHub Token

然后点击Generate token按钮，即可生成一个token

PS:新生成的 Token 只会显示一次，请妥善保存！如有遗失，重新生成即可。

配置图床

进行图床配置（绑定 GitHub Token、存储图片的仓库和目录）

  • 填写Token，点击按钮获取该用户下的仓库
  • 在仓库的下拉列表中，选择刚刚创建的仓库作为图床的仓库
  • 最后填写好额外配置信息后，点击保存即可

扩展配置

文件命名风格

标签说明例子
{YYYY}年2022
{mm}月04
{dd}日16
{sha1}文件sha1值da39a3ee5e6b4b0d3255bfef95601890afd80709
{md5}文件md5值d41d8cd98f00b204e9800998ecf8427e
{uuid}UUID4c6d0594-e224-4a23-a49c-9b09bde9b5bb
{uniqid}十三位随机字符串l21pawvk5llc9
{extension}文件扩展名.jpg、.png

接口地址

如本机与github的网络连接不是很好的话，可以使用境外服务器搭建反向代理
然后填写你搭建的反向代理域名即可
